Fictional training database

Start with the app, then learn the SQL underneath.

Practice creating, reading, updating, and deleting records in a simple appointment tracker. Every page includes a SQL guide so interns can connect each button to the database action behind it.

Do not enter real patient information. This project is only for fictional training data.

1 Patients Create fictional patient records and edit basic details. 2 Doctors Add doctors so appointments can connect to them. 3 Appointments Schedule visits, update status, and open notes.

Learning path

  1. Use the web app. Click through records before reading code.
  2. Understand tables. Learn how patients, doctors, appointments, and notes are stored.
  3. Connect buttons to CRUD. Match every action to INSERT, SELECT, UPDATE, or DELETE.
  4. Write SQL directly. Use exercises.sql when the workflow feels familiar.
